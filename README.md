# Overview

The project implements a simulation of algorithmic trading on `100ms` order-book data from [Deribit](https://docs.deribit.com/?python#json-rpc-over-websocket).

![Architecture](docs/Architecture.svg)

# Usage

```
docker compose up -d --build
```
